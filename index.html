Rules

  • Type the character displayed on the screen.
  • Follow the arrow hints:
    • Left Arrow (←): Use your left hand.
    • Right Arrow (→): Use your right hand.
    • Up Arrow (↑): Key is in the top row.
    • Down Arrow (↓): Key is in the bottom row.
    • No up/down arrow: Key is in the home row.
  • Finger numbers below the arrows indicate which finger to use:
    • 1: Thumb
    • 2: Index finger
    • 3: Middle finger
    • 4: Ring finger
    • 5: Pinky finger
